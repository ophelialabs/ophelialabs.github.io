# Services

This directory contains various service implementations that power the system.

## Contents

- `deviceMonitor.js` - Device monitoring and management service
- `groundStationService.js` - Ground station communication service
- `meshAgent.js` - Mesh network coordination service
- `nasaDataService.js` - NASA data integration service
- `quantumNetworkService.js` - Quantum network management service
- `satelliteDataService.js` - Satellite data processing service
- `supervisorAI.js` - AI supervision and coordination service
- `teleportService.js` - Quantum teleportation service
- `terminalService.js` - Terminal management service

## Purpose

The Services directory contains the core service implementations that handle various aspects of the system, from device monitoring to quantum network management. These services work together to provide a comprehensive satellite and quantum network management system.
