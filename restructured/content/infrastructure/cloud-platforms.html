Amazon Web Services (AWS)

Key Services

  • EC2: Elastic Compute Cloud for virtual machines (12-month free tier)
  • S3: Object storage for data lakes and backups
  • RDS: Managed relational databases (PostgreSQL, MySQL, MariaDB, Oracle)
  • DynamoDB: NoSQL database for high-performance applications
  • Lambda: Serverless compute for event-driven workloads
  • SageMaker: Managed machine learning platform
  • ECS/EKS: Container orchestration (ECS proprietary, EKS for Kubernetes)

Quantum Computing

  • AWS Braket: Quantum computing service with access to multiple providers
  • AWS Braket Simulator: Local quantum circuit simulation

Setup & Access

  • Free tier: 12 months of select services at no cost
  • EC2: 750 hours of t2.micro instance per month (free tier)
  • VPS: 3 months free with limited resources
  • Certain services: Extended free periods (12+ months)

Microsoft Azure

Key Services

  • Virtual Machines: Scalable computing with Windows/Linux
  • App Service: Managed platform for web and API applications
  • SQL Database: Managed SQL Server implementation
  • Cosmos DB: Globally distributed NoSQL database
  • Azure Functions: Serverless compute
  • Azure Machine Learning: Integrated ML platform
  • Azure Kubernetes Service (AKS): Managed Kubernetes
  • Azure Quantum: Quantum computing service

Quantum Computing

  • Azure Quantum: Cloud quantum computing with multiple backends
  • Q# Integration: Native Q# programming language support
  • IonQ, Rigetti, Quantinuum: Provider ecosystem access

Authentication & Access Control

  • Entra ID (formerly Azure AD): Identity and access management
  • Configure IAM roles (Owner, Editor, Viewer, Custom)
  • Enable Multi-Factor Authentication (MFA)
  • Set up billing alerts and cost management

Setup & Credits

  • Free trial: $200 USD credit for 30 days
  • Always-free services: 12+ months of select services
  • M365 Business Standard: 30-day trial available

Google Cloud Platform (GCP)

Key Services

  • Compute Engine: Virtual machines with flexible configuration
  • Cloud Run: Serverless containers
  • App Engine: Platform as a service for web applications
  • Cloud SQL: Managed SQL databases
  • Firestore/Datastore: NoSQL document databases
  • BigQuery: Data warehouse for analytics
  • Google Kubernetes Engine (GKE): Managed Kubernetes
  • Vertex AI: Unified machine learning platform

Setup & Credits

  • Free trial: $300 USD credit for 3 months
  • Always-free tier: Many services have perpetual free limits
  • Google Cloud Storage: Free egress within Google Cloud

Oracle Cloud Infrastructure (OCI)

Key Services

  • Compute: Virtual machines and bare metal instances
  • Database Cloud Service: Managed Oracle Database
  • Object Storage: S3-compatible object storage
  • Container Engine: Managed Kubernetes
  • MySQL Database Service: Managed MySQL
  • PostgreSQL Database Service: Managed PostgreSQL

Setup & Credits

  • Free trial: $300 USD credit for 30 days
  • Always-free resources: Limited but useful for development
  • Sustained use discounts: Available for committed resources

Multi-Cloud Strategy

Recommended Approach:

  • Leverage free trials/credits for evaluation (6+ months total)
  • Use AWS for broad service ecosystem and maturity
  • Use Azure for Microsoft ecosystem integration and Quantum
  • Use GCP for data analytics and machine learning
  • Implement cloud-agnostic architectures with Kubernetes
  • Use Infrastructure as Code (Terraform) for portability
