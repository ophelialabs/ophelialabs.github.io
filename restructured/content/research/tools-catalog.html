Programming Languages

Q# (Quantum Sharp)

Microsoft's quantum programming language specifically designed for quantum algorithm development and integration with classical code.

Python

Dominant language for data science, machine learning, and quantum algorithm prototyping with extensive libraries.

Java

Enterprise-grade language for scalable backend systems, microservices, and complex application architectures.

C#

.NET ecosystem integration for Windows Server, Azure cloud services, and enterprise solutions.

Rust

Systems programming language for high-performance, memory-safe applications and embedded systems.

C++

Performance-critical applications, scientific computing, and low-level system optimization.

Frameworks & Libraries

Quantum Computing

  • Qiskit (IBM): Python framework for quantum computing
  • Cirq (Google): Quantum algorithm simulation and optimization
  • Silq: High-level quantum language compiler
  • PyQuil (Rigetti): Python library for quantum program development
  • Ocean SDK: D-Wave quantum annealing tools

Backend Frameworks

  • Spring Boot: Java microservices framework
  • Flask: Lightweight Python web framework
  • .NET Aspire: Cloud-native .NET application development
  • FastAPI: Modern high-performance Python web framework
  • Django: Full-featured Python web framework

Data Processing

  • Apache Spark: Distributed data processing engine
  • Apache Hadoop: Distributed storage and processing framework
  • Pandas: Data manipulation and analysis library
  • Polars: High-performance DataFrame library
  • Dask: Parallel computing with dynamic task scheduling

Machine Learning

  • TensorFlow: Deep learning framework from Google
  • PyTorch: Dynamic deep learning framework from Meta
  • Scikit-learn: Machine learning library for Python
  • XGBoost: Gradient boosting framework
  • Keras: High-level neural networks API

Deployment & Infrastructure Tools

Containerization

  • Docker: Container platform for packaging applications
  • Podman: Container runtime alternative to Docker
  • Containerd: Lightweight container runtime

Orchestration

  • Kubernetes (K8s): Container orchestration and management
  • Docker Swarm: Docker's native orchestration
  • Nomad: Flexible workload orchestrator

Infrastructure as Code

  • Terraform: Infrastructure provisioning and management
  • Ansible: Configuration management and automation
  • Helm: Kubernetes package manager
  • CloudFormation: AWS infrastructure templates

CI/CD

  • Jenkins: Open-source automation server
  • GitLab CI: Integrated CI/CD in GitLab
  • GitHub Actions: CI/CD in GitHub
  • CircleCI: Cloud-native CI/CD platform
  • ArgoCD: Declarative GitOps for Kubernetes

Monitoring & Observability

Metrics & Monitoring

  • Prometheus: Metrics collection and time series database
  • Grafana: Visualization and analytics platform
  • DataDog: Cloud monitoring and analytics
  • New Relic: Application performance monitoring

Logging

  • ELK Stack (Elasticsearch, Logstash, Kibana): Log aggregation and analysis
  • Splunk: Enterprise logging and analytics
  • Loki: Log aggregation system for Kubernetes
  • Fluentd: Log forwarding and processing

Tracing

  • Jaeger: Distributed tracing system
  • Zipkin: Distributed tracing platform
  • Tempo: Kubernetes-native tracing backend
Note: This catalog represents the primary tools and frameworks used across research and infrastructure projects. For specific use cases or version requirements, consult individual project documentation and requirements files.
